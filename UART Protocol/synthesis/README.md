# Synthesis

## Overview

This folder contains the synthesis results generated using Xilinx Vivado.

## Files

- synthesis_uart.png

## Description

The synthesis report shows the FPGA implementation of the UART design, including:

- RTL Netlist
- Logic Optimization
- Resource Utilization
- Hardware Mapping

The synthesis confirms that the UART design is compact, efficient, and suitable for FPGA implementation.
