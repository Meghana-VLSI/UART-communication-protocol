# RTL Codes

## Overview

This folder contains the Verilog HDL source files for the UART Communication System.

## Files

- baud_generator.v
- uart_tx.v
- uart_rx.v
- uart_top.v

## Description

- **Baud Generator:** Generates the baud tick required for UART communication.
- **UART Transmitter:** Converts 8-bit parallel data into serial data.
- **UART Receiver:** Converts serial data back into parallel data.
- **UART Top Module:** Integrates all UART modules into a complete communication system.
