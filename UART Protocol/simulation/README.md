# Simulation

## Overview

This folder contains the simulation waveforms of the UART Communication System generated using Xilinx Vivado.

## Contents

- UART transmission waveform
- UART reception waveform
- schematic waveform

## Description

Simulation verifies the correct operation of UART transmission and reception using loopback testing.

The transmitted data (8'h55) is correctly received, confirming successful UART communication.
