# State Diagrams

## Overview

This folder contains the finite state machine (FSM) diagrams used in the UART design.

## Files

- uart_tx_state.png
- uart_rx_state.png

## Description

### UART Transmitter FSM

The transmitter operates through the following states:

- IDLE
- START
- DATA
- STOP

### UART Receiver FSM

The receiver operates through the following states:

- IDLE
- START
- DATA
- STOP
- DONE

These state diagrams illustrate the sequence of operations during UART communication.
